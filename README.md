# Blog Application

## 🚀 Overview
A full-stack blog platform where users can create, read, update, and delete blog posts.

## 🛠 Tech Stack
- React
- Node.js
- Express.js
- PostgreSQL
- Tailwind CSS

## ✨ Features
- User Authentication
- CRUD Blog Operations
- RESTful API Integration
- Responsive UI Design

## 📂 Project Structure
client/ → Frontend  
server/ → Backend  

## ⚙️ How to Run Locally
1. Clone the repository
2. Run `npm install`
3. Start backend server
4. Start frontend
